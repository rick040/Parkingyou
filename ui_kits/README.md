# ParkingYou rebranding prototype

High fidelity static React prototype for the ParkingYou website rebrand.

## Pages

- `#/` - homepage with hero search, featured garages, brand benefits, app callout, steps and FAQ.
- `#/garages` - searchable/filterable garage overview.
- `#/garages/philips-stadion` - example garage detail page with working reservation drawer.
- `#/abonnementen` - subscription product page with conditions, location finder and request flow.
- `#/parkingpass` - ParkingPass product page with 10/25 choice and order flow.
- `#/app` - app proposition page with phone mockup and feature flow.
- `#/over` - brand/about page with consumer and partner story.
- `#/campagnes/glow` - campaign page with event-ticket selection flow.
- `#/zakelijk` - business/garage-owner proposition.
- `#/support` - support page with working contact form and FAQ.

## Prototype behavior

- Hash routing, so it runs as a static site without a build step.
- City filters, sort control, detail pages and reservation success state.
- FAQ accordion and contact form confirmation state.
- Responsive desktop and mobile layouts.

Open `index.html` through a local web server so the JSX files can be loaded correctly.
